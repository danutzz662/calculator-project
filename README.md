# Calculator Project

A simple web-based calculator built with HTML, CSS, and JavaScript — created for The Odin Project curriculum.

**Overview:**

- A minimal, responsive calculator that performs basic arithmetic operations (addition, subtraction, multiplication, division) and handles decimal input and clearing.

**Built for:** The Odin Project (Full Stack JavaScript curriculum)

**Demo / Run locally:**

1. Open `index.html` in your browser, or serve the project folder with a static server.
2. The UI is implemented in `index.html` with styles in `style.css` and logic in `source.js`.

**Features:**

- Basic arithmetic: add, subtract, multiply, divide
- Decimal support and clear/backspace
- Responsive layout for desktop and mobile

**Technologies:**

- HTML
- CSS
- JavaScript

**Project structure:**

- `index.html` — main HTML page
- `style.css` — styles and responsive layout
- `source.js` — calculator functionality
- `README.md` — project documentation

**Installation & Development:**

- No build tools required. Clone the repo and open `index.html`.
- Optional: serve with a simple static server (e.g., `npx http-server` or `python -m http.server`) when testing fetches or CORS-sensitive features.
